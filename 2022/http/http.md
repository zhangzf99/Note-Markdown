# HTTP
## HTTP报文结构
对于 TCP 而言，在传输的时候分为两个部分：**TCP头**和**数据部分**。

而 HTTP 类似，也是 **header + body** 的结构，具体而言：

```
起始行 + 头部 + 空行 + 实体
```
### 起始行
对于请求报文来说，起始行类似下面这样：
```
GET /home HTTP/1.1
方法 + 路径 + http版本
```
对于响应报文来说，起始行一般长这样
```
HTTP/1.1 200 OK
```
响应报文的起始行也叫作**状态行**。由**HTTP版本、状态码和原因**三部分组成。
